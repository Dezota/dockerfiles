# Dremio Community Edition 

This is a docker image for [Dremio](https://github.com/dremio/dremio-oss). 
It is based on CentOS.

For more details and documentation: https://www.dremio.com/

### Usage
```
cd image
make build
cd ../compose
./run.sh
```
